# markdown-study
markdown 문법 정리

## 제목(Header)
H1~H6까지 지원한다.
```
# H1 크기 입니다.
## H2 크기 입니다.
### H3 크기 입니다.
#### H4 크기 입니다.
##### H5 크기 입니다.
###### H6 크기 입니다.
####### H7 크기 입니다. 어떻게 될까요?
```

## 제목(Header) - 결과

# H1 크기 입니다.
## H2 크기 입니다.
### H3 크기 입니다.
#### H4 크기 입니다.
##### H5 크기 입니다.
###### H6 크기 입니다.
####### H7 크기 입니다. 어떻게 될까요?


## 목록(List)
```
1. 첫번째
1. 두번째
2. 세번째
  + 인덱스와 별개로 처음부터 그냥 넘버링된다.
  - 마찮가지로 서브 목록도 +-*으로 표시 가능, 혼합해도 동일 레벨이면 같은 표시
3. 네번째
  1. 서브 넘버
    - 서브 목록 첫번째
      + 그밑에 서브 목록
      * 또 서브목록
    * 서브 목록 두번째
  2. 서브 넘버 two
```


## 목록(List) - 결과

1. 두번째
2. 세번째

3. 네번째
    1. 서브 넘버
        - 서브 목록 첫번째
        + 그밑에 서브 목록
            * 또 서브목록
            * 서브 목록 두번째
    2. 서브 넘버 two
